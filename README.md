# AWS S3

## Support

```
支持对AWS S3对象存储上传、下载、删除、查看

Usage: s3 [-c conf] [-h]

Options:
ecs S3工具，支持上传、下载、删除、查看。
author: chengken

  -c string
    	conf file (default "/app/test/.s3.yaml")
  -delete string
    	delete filename
  -download string
    	download filename
  -h	show help information
  -list
    	s3 buckets list
  -upload string
    	upload filename


```
